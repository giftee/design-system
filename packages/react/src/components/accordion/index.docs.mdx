## Props

### Accordion.Root

| Prop       | 型                    | デフォルト | 説明                                |
| ---------- | --------------------- | ---------- | ----------------------------------- |
| `size`     | `'small' \| 'medium'` | `'medium'` | アコーディオンのサイズ              |
| `disabled` | `boolean`             | `false`    | 無効状態                            |
| `open`     | `boolean`             | -          | 開閉状態（HTML details 要素の属性） |

標準的な `<details>` 要素の属性も使用できます。

### Accordion.Summary / Accordion.Content

それぞれ標準的な `<summary>`, `<div>` 要素の属性が使用できます。
